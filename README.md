# MatchMe Game 🎲

## Project Overview
The **MatchMe Game** is a fun, interactive spin-the-bottle inspired application with two engaging modes: **Single Mode** and **Pair Mode**. Players are randomly selected and challenged to complete tasks hidden inside mystery boxes, making it perfect for parties, team-building, or casual entertainment.

---

## 🎯 Game Modes
- **Single Mode**  
  Add any number of players (e.g., 11 people). The game randomly selects one player and presents them with a set of boxes (20+). When a box is opened, the player must complete the challenge or action written inside.

- **Pair Mode**  
  Add participants to two sides (e.g., boys and girls). The game randomly pairs one boy and one girl. Together, they choose a box and must complete the challenge inside as a team.

---

## ⚙️ Key Features
- Random player selection for fairness.  
- Mystery box challenges to keep gameplay exciting.  
- Two distinct modes for solo or paired play.  
- Scalable — supports large groups of players.  
- Simple, intuitive interface for quick setup.  

---

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript  
- **Logic**: Randomization algorithms for player selection and box assignment  
- **Deployment**: Works in any modern browser
